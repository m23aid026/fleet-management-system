To check testing results follow below steps:

1] Open J meter from "\apache-jmeter-5.6.3\bin\jmeter.bat" or .sh for Linux
2] Open KafkaLoadTest.jmx
3] Run test from above run option after launching kafka producer and consumer.
4] From left panel navigate to summary reports to see live results of test. 

To Test on different parameters navigate to Thread group and change parameters accordingly.
